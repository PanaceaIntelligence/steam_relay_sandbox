# steam_relay_sandbox
steam relay 를 이것저것 해보는 곳입니다.

### In Rust

0. Steamworks SDK 다운로드 및 설정.
* SDK 다운로드 주소(steam 개발자 계정만 가능)
(https://www.google.com/search?q=https://partner.steamgames.com/downloads/steamworks_sdk.zip)

* 환경변수 설정
STEAM_SDK_PATH = {SDK가 설치된 폴더 경로}

1. .devcontainer/Dockerfile 를 이용한 devcontainer 생성.
